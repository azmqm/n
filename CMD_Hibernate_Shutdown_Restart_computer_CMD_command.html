Command 1:To turn on hibernation if it is off.



powercfg -h on

Command 2: To put computer in hibernation.



shutdown /h
Or
shutdown -h

Hibernate at specific time:

If you really want to hibernate your computer after a specific time, all you need to do is to enter this command below into the cmd. i.e:

timeout /t 36000 /nobreak && shutdown /h

Now the computer will start to count down from 36000 to 0 before it will hibernate. But you should note that you can change 36000 into any number of seconds that suits you.

Shutdown Command 1:

shutdown -p -f

Shutdown Command 2:

shutdown -s -f -t 00
Note: Instead of 00, if you write 25, then it will shutdown after 25 seconds.
"-f" is to force shutdown. You can go without it.

Command to abort/cancel System Shutdown or Restart

shutdown /a

Note: There's no need to abort hibernation command because it gets aborted if you close the CMD command file.

Command to restart:

shutdown -r -f -t 00

You can write these codes in text file and change the extension .txt into .bat
After that you can execute the command by double clicking the file.
